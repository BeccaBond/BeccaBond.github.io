Interactive Topography using Fourier Transform

Summary

For our final multivariable calc project, my classmate and I created an interactive, real-time contour plot. We used a Kinect sensor to input distance values over x and y grids, then used fourier transform to approximate the multivariable function which corresponded with those values. Finally, we projected the function's contour plot in real time and enabled user control of the plot's settings.

Users were able to place objects in front of the sensor and see the contours projected onto the objects in real time. In the image below, a user places their hand in front of the sensor and visualizes the contour lines as well as gradient vectors.

Users were also able to visualize fourier transform by increasing and decreasing the radius of frequencies included in the function's approximation. When more frequencies were included, the contour plot became more accurate and more rough; when fewer frequencies were included, the contour plot became less accurate and more smooth. In the images on the right, a user increases the radius of frequencies to more accurately approximate and plot the contour of a pair of scissors.

This project was a really cool way to learn, apply and teach fourier transform. Check out our project paper below to learn more! ↓
